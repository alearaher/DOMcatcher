Welcome to DOMoCorder by Alejandro Araya
This program takes the inputs that you perform on a website and recreates them as soon as you click the "Play Macro" Button

Current limitations:
- It only records 1 website and follows only buttons and presses within the page
- It only records clicks, keystrocks and other functionality comming soon

tested on the following sites:
- https://www.macrumors.com/
- https://demoqa.com/dynamic-properties

To use this tool, follow these steps:
 0. Go to your chromium web browser of choice
 1. Navigate to the extensions menu
 2. Activate developer mode
 3. Choose Load unpacked option and point to the directory of this README!

This software was developed using Vue (with npm), Javascript assited by Gemini (Google's LLM),
